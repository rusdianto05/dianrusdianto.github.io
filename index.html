Tentang Saya

Yuk, Mengenal Saya lebih lanjut

Saya Dian Rusdianto, Mahasiswa Semester 7 Universitas Semarang Program Studi Teknik Informatika. Saya adalah seorang Web Developer yang berpengalaman dalam pengembangan website dan aplikasi web. Saya juga memiliki pengalaman dalam pengembangan website dengan menggunakan framework laravel. Saya memiliki kemampuan dalam pengembangan aplikasi dan website dengan menggunakan bahasa pemrograman PHP, Javascript, dan CSS. Saya memiliki minat yang tinggi sebagai Backend Developer dalam pengembangan website dan aplikasi.
